Privacy Policy

Effective date: June 28, 2025

This Privacy Policy explains how the Prompt Suite browser extension ("we", "our", or "the extension") collects, uses, and shares user data.

1. Data Collection and Use

The extension does not collect or store any personal information on our servers. All user data, including prompt templates and prompt history, is stored locally in the user's browser.

When using the prompt enhancement feature, prompt text is temporarily sent via secure HTTPS connection to the OpenRouter API to generate improved prompts. No data is stored by the extension or its developer.

2. Data Sharing

The extension shares prompt text only with OpenRouter for processing. During this process, standard metadata such as your IP address and browser headers may be transmitted automatically by your browser.

The developer does not access or store this information. Please refer to OpenRouter's own Privacy Policy for more details.

3. Data Storage

All user data (prompts, templates, and usage history) is stored locally using Chrome's Extension Storage API and is never transmitted to any servers under our control.

4. Security

All communications with the OpenRouter API occur over secure HTTPS connections. We do not process or retain any personal data on external servers.

5. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated effective date.
